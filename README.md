## move 游戏
//两个人玩的游戏：
//A方先发起游戏，B方在一定时间内回应（回合最大时间ROUND_TIME），A、B方在
//2*ROUND_TIME内出拳，并说出他预测的总数，获胜方要求将币存入自己的钱包；
//
//出拳：
//fist（出拳头代表0）
//palm（出手掌展开代表5）
//左右手需同时出拳
//
//无响应的异常情况：
//B方没有回应挑战，A收回自己的钱；
//
//获胜条件：
//任何一方说出的总数，是两个人出拳的总和；
//比如A出两个palm，B出一个fist，一个palm，总数应该为15，
//喊出15的一方获胜；
